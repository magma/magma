# Select

```js
import Select from '@fbcnms/ui/components/design-system/Select/Select';
```

The `Select` component supports two sizes.

`normal` size gives the menu a fixed width.

`full` size adjusts the menu's width according the its content.

export const OPTIONS = [
  {
    key: 'option_1',
    label: 'Option 1',
    value: 'wow1',
  },
  {
    key: 'option_2',
    label: 'Option 2',
    value: 'wow2',
  },
  {
    key: 'long_option',
    label: 'Option with a long label',
    value: 'wow3',
  },
];

### Normal Size

<Preview>
  <Story name="Normal Size">
    {() => {
      const classes = useStyles();
      const [selectedValue, setSelectedValue] = useState(null);

          label="Project"
