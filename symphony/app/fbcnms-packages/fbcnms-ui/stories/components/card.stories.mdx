# Card

```js
import Card from '@fbcnms/ui/components/design-system/Card/Card';
```

The `Card` component supports two variants: `standard` and `message`.

A header can be added using `CardHeader`.

Right content can be added using the `rightContent` prop.

<Preview>
  <Story name="Standard Variant">
    <Card>
      <CardHeader>Title</CardHeader>
      <Text>Content</Text>
    </Card>
  </Story>
</Preview>

### No Margins

You can control the card's inner padding.

<Preview>
  <Story name="No Margins">
    {() => {
      const classes = useStyles();
      return (
        <Card margins={CARD_MARGINS.none}>

            <Text variant="h5" color="light">
              my content here needs no margins
            </Text>
          </div>
        </Card>
      );
    }}
  </Story>
</Preview>
