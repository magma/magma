# Button

```js
import Button from '@fbcnms/ui/components/design-system/Button';
```

The `Button` component supports two variants: `contained` and `text`.

The available skins are: `primary`, `regular`, `red`, `gray`, `secondaryGray`, `orange`, `green`.

You can also supply an optional left or right icons.

### Contained Buttons

export const GrayButton = () => {
  const classes = useStyles();
  return (
    <div className={classes.grayButtonContainer}>
      <Button onClick={onButtonClicked} skin="gray">
        Gray
      </Button>
    </div>
  );
};

export const SecondaryGrayButton = () => {
  const classes = useStyles();
  return (
    <div className={classes.grayButtonContainer}>
      <Button onClick={onButtonClicked} skin="secondaryGray">
        Secondary Gray
      </Button>
    </div>
  );
}
