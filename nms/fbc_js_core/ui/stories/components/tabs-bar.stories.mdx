# TabsBar

```js
import TabsBar from '../../../ui/components/design-system/Tabs/TabsBar';
```

`TabsBar` supports three sizes: `small`, `standard` and `large`.

The `spread` prop let's you control whether the tabs will stretch to take up all the available width.

### Standard Size

<Preview>
  <Story name="Standard">
    {() => {
      const [activeTab, setActiveTab] = useState(0);
      return (
        <TabsBar
          tabs={tabs}
          activeTabIndex={activeTab}
          onChange={setActiveTab}
          spread={false}
        />
      );
    }}
  </Story>
</Preview>
