# Checkbox

```js
import Checkbox from '@fbcnms/ui/components/design-system/Checkbox/Checkbox';
```

The `Checkbox` component supports 3 states - checked, unchecked and indeterminate (`-`).

<Preview>
  <Story name="Standard">
    {() => {
      const classes = useStyles();
      const [checked, setChecked] = useState(true);
      return (
        <div className={classes.root}>
          <Checkbox
            checked={checked}
            title="Option"
            onChange={selection =>
              setChecked(selection === 'checked' ? true : false)
            }
          />
        </div>
      );
    }}
  </Story>
</Preview>
