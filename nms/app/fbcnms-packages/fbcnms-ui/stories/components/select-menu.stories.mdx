# SelectMenu

```js
import SelectMenu from '@fbcnms/ui/components/design-system/Select/SelectMenu';
```

The `SelectMenu` supports showing options with an icon or a profile picture.
It also supports disabled options and skinned options.

### Search

<Preview>
  <Story name="Search">
    {() => {
      const classes = useStyles();
      const [options, setOptions] = useState(INITIAL_OPTIONS);
      return (
        <SelectMenu
          className={classes.select}
          size="normal"
          onOptionsFetchRequested={searchTerm =>
            setOptions(
              searchTerm === ''
                ? INITIAL_OPTIONS
                : INITIAL_OPTIONS.filter(option =>
                    String(option.label)
                      .toLowerCase()
                      .includes(searchTerm.toLowerCase()),
                  ),
            )
          }
          options={options}
          onChange={value => window.alert(`Click option #${value}`)}
        />
      );
    }}
  </Story>
</Preview>
