# PopoverMenu

```js
import PopoverMenu from '@fbcnms/ui/components/design-system/Select/PopoverMenu';
```

Displays a button that when clicked -- opens a popover menu.

### Icon Button

Using PopoverMenu with an icon button

<Preview>
  <Story name="Icon Button">
    {() => {
      const classes = useStyles();
      return (
        <PopoverMenu
          className={classes.popoverMenu}
          variant="text"
          options={[
            {
              key: 'option_1',
              label: 'Option 1',
              value: '1',
            },
            {
              key: 'option_2',
              label: 'Option 2',
              value: '2',
            },
          ]}
          onChange={value => window.alert(`Clicked on item #${value}`)}>
          <MoreHorizIcon className={classes.moreIcon} />
        </PopoverMenu>
      );
    }}
  </Story>
</Preview>

### Search

Using PopoverMenu to display a dropdown menu with a search bar.

<Preview>
  <Story name="Popover with search">
    {() => {
      const classes = useStyles();
      const [options, setOptions] = useState(OPTIONS);
      return (
        <PopoverMenu
          onOptionsFetchRequested={searchTerm =>
            setOptions(
              searchTerm === ''
                ? OPTIONS
                : OPTIONS.filter(option =>
                    String(option.label)
                      .toLowerCase()
                      .includes(searchTerm.toLowerCase()),
                  ),
            )
          }
          options={options}
          onChange={value => window.alert(`Clicked on item #${value}`)}
          rightIcon={AddIcon}>
          Add Filter
        </PopoverMenu>
      );
    }}
  </Story>
</Preview>
